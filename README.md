# Alves & Tesoto Consultoria - Website

Site moderno e responsivo para a consultoria de RH & Marketing Alves & Tesoto.

## 🎯 Características

- ✨ Design premium e elegante
- 🎨 Paleta de cores em tons ouro e azul escuro
- 📱 Totalmente responsivo (mobile-first)
- ⚡ Performance otimizada
- 🤖 Seção interativa com assistente TEA (IA)
- 📊 Apresentação de resultados e métricas
- 🔗 Integração com WhatsApp

## 📋 Seções do Site

1. **Header/Navbar** - Navegação principal e logo
2. **Hero** - Headline com CTA principal
3. **Stats** - Métricas de resultados
4. **Soluções** - Serviços de RH Inteligente e Marketing Estratégico
5. **Clientes** - Logos de empresas parceiras
6. **TEA IA** - Assistente inteligente interativo
7. **CTA Final** - Chamada para ação
8. **Footer** - Informações e redes sociais

## 🛠️ Tecnologias

- HTML5
- CSS3 (com variáveis CSS)
- JavaScript Vanilla
- Font Awesome (ícones)

## 📁 Estrutura

```
site-alves-tesoto/
├── index.html
├── styles.css
├── script.js

```

## 🚀 Como Usar

1. Clone o repositório
2. Abra `index.html` no navegador
3. Customize com seus dados (WhatsApp, cores, etc)
